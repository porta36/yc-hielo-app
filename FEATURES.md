# 🌟 Características de YC HIELO

## 📋 Resumen General

YC HIELO es una aplicación de gestión integral para empresas de producción y distribución de bolsas de hielo. Proporciona control total sobre ventas, gastos, producción, clientes y pedidos con cálculos automáticos de precios y ganancias.

---

## 🎯 Características Principales

### 1. 💼 Gestión de Ventas
- ✅ Registro de ventas por cliente
- ✅ Diferentes tipos de venta (Minorista/Mayorista)
- ✅ Cálculo automático de totales
- ✅ Generación de boletas en PDF
- ✅ Códigos QR para trazar ventas
- ✅ Historial completo de transacciones
- ✅ Búsqueda y filtrado de ventas

## ⚡ Cálculos Automáticos

La aplicación calcula automáticamente:

### Ventas
```
Total Venta = Σ(Cantidad × Precio)
Ganancia Venta = Σ(Cantidad × (Precio - Costo))
Margen % = (Ganancia / Total Venta) × 100
```

### Producción
```
Total Producción = Σ(Bolsas por presentación)
Capacidad Utilizada = (Total Producción / Capacidad Máxima) × 100
```

### Rentabilidad
```
Ganancia Neta = Ingresos - Gastos
ROI = (Ganancia Neta / Inversión) × 100
Gastos Promedio = Total Gastos / Período (días)
```

- Nueva ganancia estimada: +$1,200/mes
